## Docker 1.9 21/11/15

- `docker build -e HTTP_PROXY=http://my-proxy:3128 -t my-image .` (ou encore `--build-arg`)
- `docker stats` - I/O metrics
- `docker volume` - Persistent Storage
- `docker network` - Multi-host Networking
- Plugins architecture (FS, Network, Registry storage, ...)



## Docker 1.10 - 04/02/16

- Security updates
- Content addressable image IDs
- `docker events`
- `docker update` - Live update container resource constraints
- Constraints on disk I/O

Notes :
Content addressable image IDs = Image IDs now represent the content that is inside an image, in a similar way to how Git commit hashes represent the content inside commits.
docker events : event stream



## Docker 1.12 - 12/04/16

- Complies to standard container format (`runC`)
- Engine restarts/upgrades without restarting the containers
- Labels on networks and volumes
- Better handling of low disk space

Notes :
https://blog.docker.com/2016/04/docker-engine-1-11-runc/
